## 安装

```cmd

# 使用npm
npm i mor-form-making --save
```

## 引入组件

```javascript
// 在main.js引入

import { MorFormMaking } from "mor-form-making";
import "mor-form-making/lib/mor-form-making.css";
Vue.use(MorFormMaking);
```

## 使用组件

```html
<template>
  <div>
    <mor-form-making />
  </div>
</template>
```
